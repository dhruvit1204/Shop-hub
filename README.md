
# Welcome to your ShopHub Project

## Setting Up the Project in VS Code

### Prerequisites
- [Visual Studio Code](https://code.visualstudio.com/) installed
- [Node.js](https://nodejs.org/) (v18 or later) installed
- [Git](https://git-scm.com/) installed

### Steps to Open and Run the Project

1. **Clone the Repository**
   ```bash
   git clone <YOUR_REPOSITORY_URL>
   cd shophub-project
   ```

2. **Open in VS Code**
   ```bash
   code .
   ```
   Or open VS Code and use File > Open Folder to navigate to the project directory.

3. **Install Dependencies**
   ```bash
   npm install
   ```

4. **Recommended VS Code Extensions**
   - ESLint
   - Prettier
   - TypeScript intelliSense
   - Tailwind CSS IntelliSense

5. **Run the Development Server**
   ```bash
   npm run dev
   ```

### Troubleshooting
- Ensure all dependencies are installed
- Check Node.js version compatibility
- Verify Git repository connection
